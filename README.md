# TalentHub 🚀 — Enterprise HR Management System

A complete Enterprise HR Management System built during
Weintern Internship Program 2026.

---

## 📁 Project Structure

talenthub/
├── talenthub-frontend/   → React.js Frontend
└── talenthub-backend/    → Spring Boot Backend

---

## 🖥️ Tech Stack

Frontend: React.js, Chart.js, Axios, React Router
Backend: Java, Spring Boot, Spring Security, JWT, MySQL

---

## 📋 Features
- 🔐 JWT Login & Authentication
- 👥 Employee Management (CRUD)
- 📅 Attendance Tracking
- 🏖️ Leave Apply & Approval
- 💰 Payroll Engine (Auto Calculation)
- 📊 Reports & Analytics with Charts

---

## 🚀 How to Run

### Backend
cd talenthub-backend
./mvnw spring-boot:run
Runs at: http://localhost:8080

### Frontend
cd talenthub-frontend
npm install
npm start
Runs at: http://localhost:3000
